# tubes-pbo-2-travel-online

Project ini merupakan tugas besar mata kuliah **Pemrograman Berorientasi Objek 2 (PBO 2)** dengan tema **Travel Online**.

## Deskripsi
Aplikasi Travel Online ini dibuat untuk membantu pengguna dalam melakukan pemesanan perjalanan secara online, seperti memilih paket perjalanan, melihat informasi travel, dan melakukan transaksi pemesanan.
